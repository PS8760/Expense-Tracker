# BudgetBuddy 💸

A modern expense tracker built with **Next.js**, Firebase Auth, and Tailwind CSS.

## Features

- 🔐 User Authentication (Email & Google)
- 📊 Dashboard for expenses
- 💰 Budget tracking
- 🔥 Firebase backend
- 🎨 Tailwind styling
